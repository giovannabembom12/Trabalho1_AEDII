## Objetivo do Trabalho

Este trabalho tem como objetivo compreender a **complexidade de algoritmos de busca e ordenação**, utilizando as estruturas de dados **vetor** e **lista encadeada**, medindo:
- Tempo de execução
- Média dos tempos
- Desvio padrão
- Comparação de desempenho entre algoritmos
- Geração de tabelas e gráficos com resultados

A implementação foi feita seguindo o conceito de **Tipos Abstratos de Dados (TADs)**.

---

## Tecnologias Utilizadas

- Linguagem: **C**
- Estruturas utilizadas:
  - Vetor
  - Lista Encadeada
- Algoritmos implementados:
  - Busca Sequencial
  - Busca Binária
  - Ordenação Bolha
  - Inserção Direta
  - Seleção Direta
  - Quicksort
  - Merge Sort
- Medição de tempo.
